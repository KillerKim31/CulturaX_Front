# CulturaX Frontend

Веб-платформа для прохождения онлайн-курсов. Клиентская часть реализована на **Vue 3** + **Bootstrap 5**, взаимодействует с REST API (Spring Boot) по протоколу HTTP.

## Стек технологий

| Категория | Технология |
|---|---|
| Фреймворк | Vue 3 (Composition API, `<script setup>`) |
| Сборщик | Vite 8 |
| Состояние | Pinia |
| HTTP-клиент | Axios |
| UI | Bootstrap 5, Bootstrap Icons |
| Роутинг | Vue Router 4 |

## Структура проекта

```
src/
├── assets/              # Глобальные стили (CSS)
├── components/
│   └── layout/          # Компоненты макета (Navbar)
├── router/              # Маршрутизация с guards
├── services/            # Axios-инстанс с JWT-интерцепторами
├── stores/              # Pinia-сторы (auth, courses)
├── views/
│   ├── auth/            # Авторизация (вход, регистрация, сброс пароля)
│   ├── user/            # Пользовательский интерфейс
│   ├── admin/           # Административная панель
│   └── CourseCatalogView.vue
├── App.vue
└── main.js
```

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера (http://localhost:5173)
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен-сборки
npm run preview
```

## Требования

- **Node.js** ^22.18.0 или >=24.12.0
- **npm** (в комплекте с Node.js)
- **REST API сервер** — должен работать на `localhost:8080`

## Подключение к API

Все запросы к `/api/*` проксируются Vite на `http://localhost:8080` (настроено в `vite.config.js`).

Если бэкенд запущен на другом адресе, измените секцию `server.proxy` в `vite.config.js`:

```js
server: {
  proxy: {
    '/api': {
      target: 'http://адрес-бэкенда:порт',
      changeOrigin: true,
    },
  },
},
```

## Функциональность

### Гость (неавторизован)
- Просмотр каталога курсов
- Детальная информация о курсе

### Пользователь
- Личный кабинет с прогрессом
- Запись на курсы
- Прохождение модулей и глав
- Выполнение тестов (single/multiple choice)
- Просмотр сертификатов
- Управление профилем и паролем

### Администратор
- Аналитика платформы (пользователи, курсы, записи)
- Управление пользователями (смена ролей, удаление)
- CRUD курсов, модулей и глав
- Полное редактирование структуры курса

## Маршруты

| Роут | Описание | Доступ |
|---|---|---|
| `/` | Каталог курсов | Все |
| `/login` | Вход | Гости |
| `/register` | Регистрация | Гости |
| `/forgot-password` | Сброс пароля | Гости |
| `/dashboard` | Личный кабинет | Авторизованные |
| `/my-courses` | Мои курсы | Авторизованные |
| `/course/:id` | Детали курса | Все |
| `/course/:courseId/lesson/:lessonId` | Просмотр модуля | Авторизованные |
| `/test/:id` | Прохождение теста | Авторизованные |
| `/profile` | Профиль | Авторизованные |
| `/certificates` | Сертификаты | Авторизованные |
| `/admin` | Админ-панель | Администраторы |
| `/admin/users` | Управление пользователями | Администраторы |
| `/admin/courses` | Управление курсами | Администраторы |
| `/admin/course/:id/edit` | Редактирование курса | Администраторы |
